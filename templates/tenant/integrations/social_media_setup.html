ℹ️ كيفية استخراج مفاتيح الـ API لتفعيل {{ service_name }}:

{% if service_type == 'facebook' or service_type == 'instagram' %}
  1. قم بالدخول إلى Facebook Developers وسجل دخولك.
  2. اضغط على "My Apps" ثم "Create App" لإنشاء تطبيق جديد.
  3. اختر نوع التطبيق "Business" ثم أكمل البيانات.
  4. في لوحة تحكم التطبيق، ابحث عن منتج "Messenger" (لفيسبوك) أو "Instagram Graph API" (لانستغرام) واضغط Set Up.
  5. في قسم Access Tokens، قم بربط صفحتك ثم انسخ Access Token والصقه في الحقل أعلاه.
  6. بالنسبة لـ Page ID، يمكنك العثور عليه في صفحتك على فيسبوك من خلال "About" ثم "Page Transparency".
  7. للاشتراك في الـ Webhooks لكي نستقبل الرسائل، استخدم رابط الـ Webhook التالي: {{ webhook_url }} وضع نفس "رمز التحقق من Webhook" الذي اخترته في الحقل أعلاه.
{% elif service_type == 'tiktok' %}
  1. قم بالدخول إلى TikTok for Developers.
  2. أنشئ تطبيقاً جديداً من نوع "TikTok Marketing API".
  3. اطلب الصلاحيات الخاصة بإدارة الإعلانات والمحادثات (إن وجدت).
  4. بعد الموافقة على التطبيق، ستحصل على App ID (API Key) و Access Token من لوحة تحكم التطبيق.
{% elif service_type == 'snapchat' %}
  1. ادخل إلى Snapchat Business Manager.
  2. اذهب إلى إعدادات المؤسسة (Organization Settings) ثم قسم "OAuth Apps".
  3. أنشئ تطبيقاً جديداً (Create App).
  4. بعد الإنشاء، سيظهر لك Client ID (كمفتاح API) و Client Secret (للحصول على Access Token).
  5. معرّف الحساب الإعلاني (Ad Account ID) يمكنك العثور عليه في لوحة تحكم الإعلانات الخاصة بك (يتكون من أحرف وأرقام).
{% elif service_type == 'linkedin' %}
  1. قم بزيارة LinkedIn Developers.
  2. اضغط على "Create App" وأدخل تفاصيل التطبيق واربطه بصفحة شركتك (LinkedIn Page).
  3. اذهب إلى تبويب "Auth" للحصول على Access Token عبر أداة (OAuth 2.0 tools).
  4. تأكد من تفعيل منتج "Marketing Developer Platform" من تبويب Products.
  5. معرّف المنظمة (Organization ID) موجود في رابط صفحتك على لينكدإن (الأرقام في رابط URL).
{% elif service_type == 'google_maps' %}
  1. اذهب إلى Google Cloud Console.
  2. أنشئ مشروعاً جديداً ثم اذهب إلى "APIs & Services".
  3. ابحث عن Google My Business API و Business Profile Performance API وقم بتفعيلهما.
  4. اذهب إلى "Credentials" وأنشئ OAuth 2.0 Client ID لتوليد Access Token.
  5. معرّف الموقع (Location ID) موجود في إعدادات ملف نشاطك التجاري على جوجل.
{% else %}
  1. توجه إلى إعدادات التطبيق في منصة {{ service_name }}.
  2. ابحث عن قسم المطورين (Developers) أو إعدادات الـ API.
  3. أنشئ تطبيقاً أو مفتاحاً جديداً.
  4. انسخ المفاتيح والصقها في الحقول أعلاه واضغط حفظ.
{% endif %}
